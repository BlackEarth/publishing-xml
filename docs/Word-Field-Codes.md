## PAGEREF Bookmark [Switches]
* \\h — hyperlink to marked paragraph
* \\p — insert relative position of the marked paragraph
* \\* — format switch
	- Arabic
	- alphabetic
	- ALPHABETIC
	- roman
	- ROMAN
	- Ordinal
	- CardText
	- OrdTex
	- Hex
	- DollarText
	- Upper
	- Lower
	- FirstCap
	- Caps
* \\# — numeric picture, e.g.
	- "#,##0"
	- "#,##0.00"
	- "$#,##0.00;($#,##0.00)"

## TOC [Switches]
* \\a — build table of figures w/o caption label and number
* \\b Bookmark — bookmark of document section from which to build TOC
* \\c Label — build table of figures of the given label
* \\d Separator — define separator between sequence and page numbers
* \\f — build TOC with TC entries instead of outline levels
* \\l Level — define the TC entries field level used to build TOC
* \\n build a TOC or range of entries in a TOC without page numbers
* \\o [Range] – build a TOC using outline levels instead of TC entries
* \\p Separator — define separator between table entry and page number
* \\s [SeqType] — build TOC using sequence type
* \\t [StyleNames] — build TOC using style names rather than outline levels 
* \\w — preserve tab characters within table entries
* \\x — preserve newline characters within table entries
